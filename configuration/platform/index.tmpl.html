MICO Prototype Platform

This server has been configured with an installation of the MICO Prototype Platform for research and development purposes. This page gives basic information and entry points for the installation.

Login

Use the following login details for accessing the server:

Username: $USERNAME
Password:

Services

The following services are accessible via a web browser:

  • Apache Marmotta (Metadata Storage): http://$HOSTNAME:8080/marmotta/
  • MICO Broker (Service Orchestration): http://$HOSTNAME:8080/broker/
  • Extractor configuration (Extraction pipeline): http://$HOSTNAME:8080/mico-configuration/
  • RabbitMQ (Event Communication): http://$HOSTNAME:15672/

Documentation

The following documentation is available:

  • MICO Java Client API
  • MICO C++ Client API
